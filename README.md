## Healthcare Facilities in Nigeria — Analysis
## Overview
This project analyses gaps in healthcare service delivery across different facility types in Nigeria. It explores trends, identifies underserved areas, and provides data-driven recommendations to improve healthcare access and quality.
## Objectives
a)	Identify gaps in healthcare service delivery across facility types
b)	Analyse distribution of healthcare facilities across Nigeria
c)	Provide insights and strategic recommendations
## Tools Used
a)	Python — data cleaning and analysis
b)	Pandas — data manipulation
c)	Matplotlib / Seaborn — data visualization
## Key Findings
a)	Disparities exist across facility types in service availability
b)	Rural areas show significantly lower facility coverage
c)	Certain facility types consistently underperform in service delivery
## Limitations
a)	Dataset may not capture the most recent facility updates or closures
b)	Analysis is based on available reported data; unreported facilities are not accounted for
c)	Regional data gaps may affect the accuracy of findings in certain states
d)	The dataset does not capture quality of care, only facility availability
